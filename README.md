# Movie Explorer App

## Description

Movie Explorer is a responsive web application that allows users to search and browse movies through a clean and modern interface.

## Features

* Search movies by title
* Responsive movie card layout
* Movie poster display
* Ratings and release dates
* Mobile-friendly UI

## Technologies Used

* HTML
* CSS
* JavaScript
* GitHub Pages

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/movie-explorer.git
```

2. Open the project folder:

```bash
cd movie-explorer
```

3. Open `index.html` in your browser.

## Deployment

The project is deployed using GitHub Pages.
